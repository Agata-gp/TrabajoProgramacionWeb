# DreamShop 🌙

Una tienda web de videojuegos, tarjetas regalo y accesorios gaming moderna y responsiva.

## ✨ Características

- **Diseño Responsivo**: Se adapta perfectamente a dispositivos móviles, tablets y escritorio
- **Interfaz Moderna**: Gradientes atractivos y animaciones suaves
- **Múltiples Categorías**:
  - 🎮 Videojuegos
  - 🎁 Tarjetas Regalo Digitales
  - 👕 Skins y Cosméticos
  - 📦 Cajas Sorpresa
  
- **Carrito de Compras**: Sistema integrado para gestionar compras
- **Navegación Intuitiva**: Menú claro y fácil de usar
- **Productos Reales**: Catálogo completo con precios y descripciones

## 📁 Estructura del Proyecto

```
DreamShop/
├── index.html                    # Página principal
├── producto_videojuego.html     # Catálogo de videojuegos
├── producto_tarjetas.html       # Tarjetas regalo
├── producto_skins.html          # Skins y cosméticos
├── producto_cajas.html          # Cajas sorpresa
├── productos.html               # Catálogo completo
├── carrito.html                 # Página del carrito
├── style.css                    # Estilos principales (responsivo)
├── style-carrito.css            # Estilos opcionales carrito
├── style-productos.css          # Estilos opcionales productos
├── reset.css                    # Reset de estilos CSS
├── javascript.js                # Lógica del logo canvas
├── Nova_Mono/                   # Fuente personalizada
│   └── NovaMono-Regular.ttf
└── README.md                    # Este archivo
```

## 📱 Responsividad

El sitio es totalmente responsivo con breakpoints en:
- Escritorio: 1200px+
- Tablet: 768px - 1199px
- Móvil: < 768px

## 🚀 Características de Desarrollo

### Canvas Logo
El archivo `javascript.js` dibuja un logo dinámico en canvas:
- Luna con estrellas
- Escalado automático según la página
- Estilos personalizados

### Productos
Cada página de productos incluye:
- Grid responsivo de productos
- Tarjetas con hover effects
- Información clara de precios
- Botones de acción

### Navegación
- Menú principal consistente en todas las páginas
- Enlaces contextuales en sidebars
- Carrito siempre accesible
